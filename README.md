# Color Palette Manager

Create, manage, and organize color palettes directly in Raycast, with AI-powered color generation.

## Features

- **Save Color Palette** — build and store custom palettes with up to 15 hex colors, a name, an optional description, a light/dark mode tag, and reusable keywords for organization.
- **Manage Color Palettes** — browse and search your saved palettes, see all colors and metadata in a detail pane, and edit, duplicate, or delete any palette.
- **Create Colors with AI** — generate a color set from a text prompt with adjustable creativity (Low / Medium / High / Maximum), preview the swatches, and save the selection as a new palette.
- **Multiple export formats** — copy a single color or an entire palette as JSON, CSS classes, CSS variables, or plain text.
- **Coolors.co integration** — copy a shareable Coolors.co link for any saved palette.

## Commands

| Command | What it does |
| --- | --- |
| Save Color Palette | Opens the palette form for creating a new palette. |
| Manage Color Palettes | Lists every saved palette with full editing actions. |
| Create Colors with AI | Generates colors from a prompt, then hands the selection off to the save form. |

## Requirements

- macOS 12 or later.
- A **Raycast Pro** subscription is required for the *Create Colors with AI* command (it uses Raycast's built-in AI). The other two commands work on any Raycast plan.
